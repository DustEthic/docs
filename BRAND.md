# Identité visuelle DustEthic

Date : 2026-06-13

## Direction

DustEthic doit être perçu comme :

- moderne ;
- Web3 mais lisible ;
- technique mais humain ;
- sobre sur les promesses ;
- orienté preuve.

## Message court

```text
DustEthic
Poussières crypto → dons ONG vérifiables
Standard ouvert - sans token - frais visibles - preuves publiques
```

## Usage du logo

Le symbole représente un flux de dusts qui se transforme en graine/don vérifiable.

À utiliser pour :

- avatar GitHub ;
- avatar Discord ;
- avatar Bluesky ;
- favicon ;
- social preview ;
- documents publics.

## Fichiers source

- `brand/dustethic-mark.svg` : symbole carré.
- `brand/dustethic-logo-wordmark.svg` : logo horizontal.

## À éviter

- graphisme financier agressif ;
- promesse de rendement ;
- vocabulaire de token launch ;
- logo trop ONG classique ;
- logo qui fait exchange crypto.

## Couleurs de travail

- violet : énergie Web3 ;
- cyan : transparence et preuve ;
- vert : don et utilité ;
- fond sombre : profondeur technique.

## Bio courte pour profils

```text
Open standard for turning crypto dust into verifiable NGO donations. Phase 0. No token. No fundraising. Transparent fees. Public proofs.
```

## Bio française

Standard ouvert pour transformer les dusts crypto en dons vérifiables pour des ONG. Phase 0. Aucun token. Aucun fonds collecté par DustEthic.
