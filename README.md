# DustEthic - Documentation

Documentation publique du standard ouvert **DustEthic**.

**Statut : Phase 0 - conception** · **Version : v0.1-draft** · **Licence : CC BY 4.0**

DustEthic est une spécification ouverte décrivant comment un wallet crypto peut transformer les soldes résiduels (dusts) d'un utilisateur en don direct à une ONG, sans intermédiaire, sans commission et sans token. Toute la documentation est publique, en français et en anglais, et ouverte aux propositions par Issue ou Pull Request.

## English

Public documentation for the **DustEthic** open standard. Phase 0 - design, v0.1-draft, CC BY 4.0.

DustEthic is an open specification describing how a crypto wallet can turn a user's residual balances (dust) into a direct donation to an NGO, with no intermediary, no commission and no token.
