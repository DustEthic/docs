# Contribuer à DustEthic

Merci de contribuer avec prudence : DustEthic touche aux dons, aux actifs numériques, aux permissions wallet et à la confiance publique.

## Bon format de contribution

Avant de proposer une solution, documente :

- le problème exact ;
- l'utilisateur concerné : donateur, ONG, wallet, relayeur, auditeur ;
- l'hypothèse technique ;
- le risque principal ;
- la preuve ou le test nécessaire ;
- les sources.

## Ce qui est bienvenu

- clarification du standard ;
- schémas d'intention ;
- modèles de preuve ;
- revues de risques ;
- UX de signatures ;
- contraintes ONG ;
- veille technique sourcée ;
- corrections de formulation ;
- traductions prudentes.

## Ce qui n'est pas accepté

- promesse de rendement ;
- promotion spéculative ;
- collecte active ;
- code présenté comme prêt en production sans audit ;
- dépendance à un acteur unique sans alternative ;
- affirmation technique sans source.

## Langues

Le projet accepte le français et l'anglais. Le français peut rester la langue de travail initiale ; l'anglais sera utile pour l'adoption technique.

## Sécurité

Ne publie pas de données privées, signatures sensibles, clés, secrets, endpoints internes ou informations permettant une exploitation.

Pour un problème de sécurité, consulte `SECURITY.md`.
